
# 电商筛选主键套件 (SKU & SPU Filter Suite)

这是一个专为跨境电商管理后台设计的筛选主键套件，包含 `SKUFilter` 和 `SPUFilter` 两个核心组件。

## 1. SKUFilter (单品级筛选)
*   **用途**：针对具体的最小库存单位进行精准筛选。
*   **展示项**：SKU 编号、所属 SPU 编号、多语言名称、**单品实时库存**。
*   **视觉**：采用蓝色系主题。

## 2. SPUFilter (产品级筛选)
*   **用途**：针对产品系列或父级型号进行批量批量筛选。
*   **展示项**：SPU 编号、多语言名称、**关联 SKU 总数**。
*   **视觉**：采用紫色系主题。

## 2024 更新特性
*   **全宽展开面板**：当已选数量超过 `maxTagCount` 时，鼠标悬停于 `+N` 指示器可触发**全宽展开面板**。该面板与筛选框底部对齐并保持 4px 间距，支持快速预览及删除已选标签。
*   **层级管理优化**：引入严谨的 Z-Index 架构（预览卡片 z-400 > 展开面板 z-300 > 搜索下拉框 z-200），确保复杂交互下 UI 不遮挡。
*   **智能侧边预览**：大图预览卡片会根据视口空间自动选择在左侧或右侧弹出，彻底解决侧边栏遮挡问题。
*   **全维度搜索高亮**：支持按编号、中文、英文实时搜索，并在结果中精准高亮匹配字符。

## 共同特性
*   **窄容器优化**：完美适配 250px-300px 宽度容器，支持标签自动收纳。
*   **四种逻辑模式**：包含、不包含、为空、不为空。
*   **高性能**：内置搜索防抖、列表无限滚动以及已选数据名称缓存。
